# CP Tracker Dashboard

A modern dashboard to track DSA and Competitive Programming problems.

## Features

- Add Problems
- Delete Problems
- Edit Problems
- Search Problems
- Filter by Topic
- Dark Mode
- Responsive Sidebar
- Charts Dashboard
- Local Storage Support

## Tech Stack

- HTML
- CSS
- JavaScript
- Chart.js

## Future Improvements
